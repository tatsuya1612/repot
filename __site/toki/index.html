Franklin.jlとは

実装

実装方針

Franklin.jlは拡張性が高いため、独自の機能を簡単に追加できます。今回はAdmonitionsを実装し、ドキュメントの可読性と視覚的な訴求力を高めることを目的としました。

実際のコード

Admonitionsの実装コードは以下の通りです:

using Franklin

# Admonition blocks
franklin_plugin_admonition() = Pair{Regex,Template}[
    r"^@@(warning|note|tip)$" => """
    <div class="admonition \$1">
    <p class="admonition-title">\$1</p>
    \$body
    </div>
    """
]

Franklin.plugins["admonition"] = franklin_plugin_admonition

ユーザーが@@warning、@@note、@@tipのようにブロックを記述すると、適切なHTMLタグが生成されます。

課題

現時点では基本的なAdmonitionsの実装ができましたが、以下のような課題が残っています:

  • カスタムアイコンの追加

  • 複数行のAdmonitionsへの対応

  • ユーザー定義のAdmonitionタイプの追加

今後はこれらの課題に取り組み、Franklin.jlの拡張性をさらに高めていきたいと考えています。
